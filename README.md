# Nitori AI 3D Sample

React + Three.js demo that renders a rotating 3D cube in the browser.

## Requirements

- Node.js 18+

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## What to edit

- The Three.js scene lives in `src/App.tsx`.
- Layout and styling live in `src/App.css` and `src/index.css`.
